# Installation

## Stable release

To install NYC-Taxi, run this command in your terminal:

```sh
uv add python-boilerplate
```

Or if you prefer to use `pip`:

```sh
pip install python-boilerplate
```

## From source

The source files for NYC-Taxi can be downloaded from the [Github repo](https://github.com/AvanindraBose/python_boilerplate).

You can either clone the public repository:

```sh
git clone git://github.com/AvanindraBose/python_boilerplate
```

Or download the [tarball](https://github.com/AvanindraBose/python_boilerplate/tarball/master):

```sh
curl -OJL https://github.com/AvanindraBose/python_boilerplate/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd python_boilerplate
uv pip install .
```
